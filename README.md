# \# 市内人気エリア ランダム抽出ツール（地名コピー）

# 

# 市区町村名を入力すると、\*\*人口上位の町名からランダムで候補を表示\*\*し、  

# \*\*クリック一発で地名だけをコピー\*\*できるシンプルなWebツールです。

# 

# 引っ越し検討、物件探し、記事ネタ作り、アイデア出しなどに使えます。

# 

# 🔗 公開URL  

# https://psycho0617-create.github.io/city-area-randomizer/

# 

# ---

# 

# \## 🔍 できること

# 

# \- 市区町村名を入力（部分一致OK）

# &nbsp; - 例：川越 / 那覇 / 札幌

# \- 人口上位エリアからランダム抽出

# \- 表示件数を選択可能

# \- \*\*地名だけをワンクリックコピー\*\*

# \- コピー履歴を自動保存（ブラウザ内）

# \- スマホ・PC両対応

# 

# ---

# 

# \## ⚙️ 設定項目の説明

# 

# \### 抽出候補数

# 人口上位から「どこまでを候補にするか」を選びます  

# （例：10 → 人口上位10町の中からランダム）

# 

# \### 表示件数

# 実際に画面に表示する件数です

# 

# \### 市名を付けて表示

# コピー時は町名のみ  

# 表示上は「市名＋町名」で確認できます

# 

# ---

# 

# \## 📋 表示について

# 

# \- 表示：人口付き（参考情報）

# \- コピー：\*\*地名のみ\*\*

# \- 人口データは \*\*e-Stat（政府統計）\*\* を利用
